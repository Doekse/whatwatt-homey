whatwatt приносит энергетическую аналитику в реальном времени в Homey. С whatwatt Go — простым, универсальным устройством для умных счетчиков — вы получаете безопасный, мгновенный доступ к данным о потреблении. Используйте эти данные для адаптации поведения, мониторинга и контроля нагрузок, получения своевременных предупреждений, оценки производительности и распределения затрат, чтобы повысить локальную энергоэффективность и снизить счета.

Построенный как открытая, масштабируемая экосистема, whatwatt подходит для домохозяйств, предприятий и виртуальных организаций. Делая данные в реальном времени доступными для всех, whatwatt помогает избежать пиков спроса способом, который имеет смысл для пользователей и поставщиков, поддерживая безопасное энергоснабжение и более широкий энергетический переход в Швейцарии.
