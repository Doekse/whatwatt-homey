whatwatt lleva insights energéticos en tiempo real a Homey. Con whatwatt Go — un dispositivo simple y universal para medidores inteligentes — obtienes acceso seguro e instantáneo a tus datos de consumo. Usa estos insights para adaptar el comportamiento, monitorear y controlar cargas, recibir alertas oportunas, evaluar el rendimiento y asignar costos, para que aumentes la eficiencia energética local y reduzcas las facturas.

Construido como un ecosistema abierto y escalable, whatwatt se adapta a hogares, empresas y organizaciones virtuales por igual. Al hacer que los datos en tiempo real estén disponibles para todos, whatwatt ayuda a evitar picos de demanda de una manera que tiene sentido para usuarios y proveedores, apoyando un suministro energético seguro y la transición energética más amplia en Suiza.
